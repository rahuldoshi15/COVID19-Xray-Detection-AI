# Dataset

This project uses the **COVID-19 Radiography Database**, which is publicly available on Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/tawsifurrahman/covid19-radiography-database

Due to GitHub file size limitations and the dataset license, the dataset is not included in this repository.

## Dataset Classes

- COVID-19
- Normal
- Lung Opacity
- Viral Pneumonia

After downloading the dataset, extract it into the `dataset/` directory before running the notebooks.
